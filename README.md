Project Objective
The purpose of this project is to:
Practice real-world data cleaning
Apply business analysis thinking
Build interactive dashboards
Extract actionable insights
Strengthen Excel skills for Data Analysis
--
Business Value
This project demonstrates how raw booking data can be transformed into actionable business intelligence that helps:
Reduce cancellations
Optimize pricing strategy
Improve revenue channels
--
Dataset Information
The dataset contains hotel booking records including:
Booking status (Canceled / Not Canceled)
Lead time
Arrival year & month
Room type
Booking channel (Online / Offline / Corporate / Aviation / Complementary)
Average Daily Rate (ADR)
Customer segmentation
--
Data Cleaning Process
Before analysis, the dataset went through several cleaning steps:
Removed duplicates
Handled missing values
Standardized categorical values
Checked data types and formatting
Verified logical consistency (e.g., cancellation vs booking status)
This ensured data accuracy and reliability for reporting.
--
Key KPIs
From the analysis, the following KPIs were calculated:
Total Bookings: 36,275
Total Cancellations: 11,885
Room Average Price (ADR): 103.42

--
These KPIs provide a high-level performance summary of the hotel.
